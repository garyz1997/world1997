Calculator Instructions

Instructions

Commands Example Meaning
start num Start 10 Enters 15 into the calculator and overwrites previous value
add num add 5 adds 5
sub num sub 2 subtracts 2
div num div 2 divides by 2
! factorial gives the product of the number in the calculator and all the integers before it
mult num mult 3 multiplies by 3
abs abs absolute value of number in calculator
sqrt sqrt calculates the square root of the number in the calculator
expt num expt 3 raises the number in the calculator to 3
print print prints the result


Short Instructions & Tips
* Always start with the start # function
* Always remember to print
* Every function has its own line.
* You can only have one start and one print function each time but you can do any number of functions
